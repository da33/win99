# 自動上傳 LINE 圖文選單

## 📋 使用步驟

### 1. 準備背景圖片
- 檔名：`richmenu-image.png`
- 尺寸：2500x1686px
- 放在專案根目錄

### 2. 確認環境變數
確保 `.env` 有設定：
```
LINE_CHANNEL_ACCESS_TOKEN=你的token
```

### 3. 執行上傳
```bash
node upload-richmenu.js
```

## 🎨 背景圖片設計規格

### 按鈕區域座標
```
上排（高度 843px）：
- 左：立即註冊領168 (0, 0, 1250, 843)
- 右：我要儲值 (1250, 0, 1250, 843)

下排（高度 843px）：
- 左：我要提款 (0, 843, 833, 843)
- 中：查看優惠 (833, 843, 834, 843)
- 右：聯繫客服 (1667, 843, 833, 843)
```

### 建議配色
- 背景：深藍 #0d2846
- 文字：白色 #FFFFFF
- 強調：金色 #FFD700

## ⚠️ 注意事項
- 圖片必須是 PNG 或 JPG
- 檔案大小 < 1MB
- 確保 Channel Access Token 有效
