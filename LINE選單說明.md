# RG富遊 LINE@ 快捷選單設計

## 📱 選單配置（2x3 格式）

### 上排（3個按鈕）
1. **🎮 立即註冊** - 連結到官網註冊頁面
2. **💰 儲值教學** - 觸發「如何儲值」問答
3. **💸 提款說明** - 觸發「如何提款」問答

### 下排（3個按鈕）
4. **🎁 優惠活動** - 觸發「有什麼優惠」問答
5. **👤 真人客服** - 轉接真人客服
6. **🌐 官方網站** - 連結到官網

## 🎨 設計建議

### 背景圖片尺寸
- 寬度：2500px
- 高度：1686px
- 格式：JPG 或 PNG
- 大小：< 1MB

### 顏色建議（富遊風格）
- 主色：深藍色 #0d2846
- 輔色：金色 #FFD700
- 文字：白色 #FFFFFF

## 📝 設定步驟

1. 登入 LINE Official Account Manager
2. 進入「圖文選單」
3. 點擊「建立」
4. 上傳背景圖片（2500x1686px）
5. 設定 6 個區塊的動作
6. 儲存並啟用

## 🔗 按鈕動作設定

已在 `line-richmenu.json` 中配置好所有按鈕的座標和動作。
